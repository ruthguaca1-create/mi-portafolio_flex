#  Mi Portafolio flex

Este proyecto corresponde a mi portafolio personal, desarrollado como parte de mi proceso de formación en el programa **Análisis y Desarrollo de Software (ADSO)** del **SENA**.

El sitio web presenta información sobre mí, mis conocimientos, mis objetivos y algunos de los proyectos que he desarrollado durante mi formación.

---

##  Características

- Diseño moderno y responsivo.
- Menú de navegación.
- Sección de presentación.
- Información personal.
- Objetivos profesionales.
- Proyectos con enlaces a GitHub.
- Sección de contacto.
- Botón para volver al inicio de la página.
- Desarrollo utilizando HTML, CSS y JavaScript.

---

##  Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Visual Studio Code
- Git
- GitHub

##  Cómo ejecutar el proyecto

1. Clonar el repositorio.

```bash
git clone https://github.com/ruthguaca1-create/mi-portafolio_flex.git
```

2. Abrir la carpeta del proyecto en Visual Studio Code.

3. Ejecutar el archivo **índice.html** o abrir el proyecto con **Live Server** para una mejor experiencia.
